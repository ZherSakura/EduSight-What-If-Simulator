## Backend Integration Requirements

### Student Profile API

The frontend expects the `GET /students/:id` endpoint to return the student's profile information together with the latest risk assessment.

The values `currentScore` and `currentRisk` should be generated by the backend using the ML Engine and Risk Engine based on the student's latest data. The frontend will only display these values and will not perform any risk calculations locally.

Required fields:

```json
{
  "id": "string",
  "name": "string",
  "form": "string",
  "school": "string",
  "attendance": "number",
  "grades": "number",
  "counselling": "number",
  "welfare": "number",
  "currentScore": "number",
  "currentRisk": "string"
}
```

---

### Simulation API

The frontend sends the simulation inputs to the backend through `POST /simulate`.

Request body:

```json
{
  "studentId": "string",
  "inputs": {
    "attendance": "number",
    "grades": "number",
    "counselling": "number",
    "welfare": "number"
  }
}
```

The backend should process the request using the ML Engine, Risk Engine, and Recommendation Engine, then return the simulation result.

Required response fields:

```json
{
  "baselineScore": "number",
  "projectedScore": "number",
  "riskLevel": "string",
  "riskLabel": "string",
  "scoreChangeText": "string",
  "dropoutProbability": "number",
  "insightText": "string",
  "narrative": "string",
  "weights": {
    "attendance": "number",
    "academic": "number",
    "socioeconomic": "number",
    "family": "number"
  },
  "recommendations": [
    {
      "category": "string",
      "urgency": "critical | high | medium | routine",
      "title": "string",
      "description": "string",
      "expected_impact": "string"
    }
  ]
}
```

---

### Engine Responsibilities

| Component             | Responsibility                                                                                                             |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------- |
| ML Engine             | Generate predictive outputs and risk-related features from student data                                                    |
| Risk Engine           | Calculate current risk score, projected risk score, risk level, dropout probability, factor weights, and narrative summary |
| Recommendation Engine | Generate intervention recommendations and urgency levels                                                                   |
| Frontend              | Display all values returned by the backend and submit simulation inputs                                                    |

### Frontend Behaviour

* The frontend does not calculate risk scores, probabilities, weights, or recommendations.
* All displayed results should originate from the backend response.
* If the backend is unavailable during development, mock data will be used as a temporary fallback.
* Once the backend APIs are available, the frontend will automatically display the values returned by the cloud services.
